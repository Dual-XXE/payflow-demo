## ✨ Features

- **Working Paystack Checkout** — Real payment popup with test mode
- **Multiple Pricing Plans** — Basic, Pro, Premium tiers with interactive selection
- **Live Transaction Log** — Tracks successful payments in real-time
- **Payment Receipt** — Detailed success page with transaction reference
- **Responsive Design** — Beautiful on mobile and desktop
- **Code Showcase** — Displays clean integration code examples
- **Multi-Gateway Display** — Showcases Paystack, Stripe, Flutterwave expertise
- **Smooth Animations** — Scroll-based fade-in effects

## 💳 Payment Flow

```
User selects plan (Basic/Pro/Premium)
  → Checkout page shows order summary
  → User enters email
  → Clicks "Pay Now with Paystack"
  → Paystack popup opens
  → User enters card details
  → Payment processed
  → Success page with receipt
  → Transaction added to log
```

## 🔒 Security Notes

- Uses Paystack test mode (no real charges)

- All card data handled by Paystack — never touches your server
